# AI ChatBot

A simple and interactive AI ChatBot application built with React, Vite, and Tailwind CSS. It integrates with the Google Gemini API to answer your questions and solve your doubts seamlessly.

## Features
- Real-time chat interface.
- Powered by Gemini API (`gemini-3-flash-preview`).
- Responsive and modern UI styled with Tailwind CSS.

## Getting Started

1. Set up your `.env` file in the `src` directory with your Gemini API URI:
   ```env
   VITE_GEMINI_API_URI=your_api_uri_here
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
